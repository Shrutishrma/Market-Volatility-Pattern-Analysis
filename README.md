# 📈 Market Volatility Pattern Analysis

This repository contains a data science project focused on analyzing financial market fluctuations. I built this notebook to investigate historical market data, calculate risk metrics, and visualize the patterns that emerge during periods of high economic volatility.

### **What I Built (The Simple Version)**
I created an analytical workspace to understand how stock markets behave over time:
* **Data Processing:** Cleaned and structured raw historical financial data so it could be analyzed mathematically.
* **Volatility Tracking:** Calculated key financial metrics, such as daily returns and rolling variance, to measure exactly how "bumpy" the market is on any given day.
* **Trend Visualization:** Plotted clear, interactive charts to visually identify major market events, crashes, and stabilization periods.
* **Pattern Recognition:** Used statistical techniques to find underlying trends that aren't obvious just by looking at raw price numbers.
